Welcome To Elderway

Elderway is a playground for my apps and things I like. It may change a considerable amout especially in the beginning because I am learning as I go.

The main feature at this point is the bible quiz schedule generator that I am working on. I created a desktop version of this previously. You can download the source code as well as portable Windows version.

I wanted to work on a web version. As of current you can add,edit and delete teams. You can even generate a schedule in Microsoft excel format. What is missing vs the desktop version is the ability create a PDF schedule and to change some settings. Like how many quizzes, length of lunch, title of the schedule and start time. These things will be added over time. The big advantage for creating the web version is it being cross platform.
